  • [教學] 如何在 WIN10 中把 WIN7 時代的 踩地雷、接龍 等遊戲裝回來

分享: 

人生就是這麼難 !!! 沒有遇到疫情的當下還真的沒想到要玩這些遊戲

有經歷過 WINDOWS 7 以前(包含) 版本的人多少都會玩過這些經典小遊戲

踩地雷、經典接龍、新接龍 ... 等

但走進 WINDOWS 10 的時代，這些遊戲就不再被系統預載了

想玩就得到 Microsoft Store 中去下載

雖然多數還是免費的，但介面的感覺就總是差了那麼一些熟悉感

所以還是讓我們安裝回 WIN7 版本的遊樂場吧 !!!

 


  

在此我們要感謝 WINAERO.com 為大家所做的一切

 

想安裝 WIN7 遊樂場 的朋友請點我下載安裝檔

 

下載完成並解壓縮之後，直接執行安裝檔

如果你看到下圖

請選擇 "是"

 

它會自動依您的系統語系版本來選擇安裝後的語言版本

一般來說會是下圖中的 繁體中文

如果您看到的不是如下圖中那樣子，但你又很堅持要安裝中文版

就請自行在下拉選單中選擇 Chinese (Traditional)

 

然後就開始安裝吧

安裝內容將會包含底下這幾款遊戲

  • Solitaire
  • Spider Solitaire
  • Minesweeper
  • FreeCell
  • Hearts
  • Chess Titans
  • Mahjong Titans
  • Purble Place
  • Internet Spades
  • Internet Checkers
  • Internet Backgammon

安裝中會自動幫你從 WINDOWS 元件中啟動

未來不想要再見到這些可以到元件中去取消便可

 

安裝完成之後就可以在程式集中看到啦
